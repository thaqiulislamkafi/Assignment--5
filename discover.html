Question-1: What are the different ways to select an element in the DOM?

There are 5 ways basically 4 ways to select an element in the DOM such as
1. "getElementById" which is use to select one element with a unique ID.
2. "getElementsByClassName" which is use to select all elements with the same class.
3. "getElementsByTagName" which is use to select all elements with the same tag.
4. "querySelector" which is use to select the first matching element.
5. "querySelectorAll" which is use to select all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets the HTML content inside an element and it Includes all HTML tags and text, on ther hand innerText gets only the visible text inside an element and Ignores hidden elements. Besides, textContent gets all the text inside an element, including hidden text and ignores HTML tags and styling.

Question-3: What is event delegation in the DOM?

Event delegation refers to use a single event listener into a parent element to handle events for its child elements.It means we need not to add event listeners to each child , we add one into the parent

Question-4: What is event bubbling in the DOM?

In the DOM, Event bubbling refers an event starts from the target element and "bubbles up" through its parent elements according to the hierarchy

Question-5: How do you create, add, and remove elements using JavaScript?

we use document.createElement() to make a new element, use appendChild() to add the element to the DOM, use removeChild() or remove() to delete an element
